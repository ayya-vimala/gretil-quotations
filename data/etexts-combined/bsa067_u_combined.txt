   Nāgārjuna: Bhavasaṅkrāntiparikathā	NAga arjuna: Bhava saFkrAnti parikaTA
   namaḥ kumārabhūtāya mañjuśriye /	namaH kumAra BUtAya maQju Sriye /
   bhāvānna jāyate 'bhāvo nābhāvādapi jāyate /	BAvAt na jAyate aBAvaH na aBAvAt api jAyate /
   bhāva utpaddyate nityaṃ bhābo bhrāntiḥ khapuṣpavat // 1 //	BAvaH utpaddyate nityam BAbaH BrAntiH Ka puzpa vat // 1 //
   sati dharme nabhastulye khatulyaṃ jāyate param /	sati Darme naBaH tulye Ka tulyam jAyate param /
   pratītya sarva khasamaṃ bhāvastasmādabhāvavān // 2 //	pratItya sarva Ka samam BAvaH tasmAt aBAvavAn // 2 //
   sbabhāvataḥ karma nāsti heturnāsti phalaṃ na ca /	sbaBAvataH karma na asti hetuH na asti Palam na ca /
   na viddyata idaṃ sarbaṃ loko nāsti na bhañjanam // 3 //	na viddyate idam sarbam lokaH na asti na BaQjanam // 3 //
   anutpannaśca yo bhāvaḥ paraṃ (sa) janayetkatham /	anutpannaH ca yaH BAvaH param (sa) janayet kaTam /
   lokaḥ prathamato 'jātaḥ kenāpi na hi nirmitaḥ /	lokaH praTamatas ajAtaH kena api na hi nirmitaH /
   somasiṃhapurītulyo loko bhramyatyanarthake // 4 //	soma siMhapurI tulyaH lokaH Bramyati anarTake // 4 //
   loko vikalpadutpanno vikalpaścittasaṃbhavaḥ /	lokaH vikalpat utpannaH vikalpaH citta saMBavaH /
   cittaṃ hi kāyāśrayakaṃ tasmātkāyo vicāryate // 5 //	cittam hi kAya ASrayakam tasmAt kAyaH vicAryate // 5 //
   rūpaṃ śūnyaṃ vedanā niḥsvabhāvā saṃjñā nāste nāsti saṃskārabhāvaḥ /	rUpam SUnyam vedanA niHsvaBAvA saMjQA na Aste na asti saMskAra BAvaH /
   bhūtaṃ hitvā cittacaitte ca nastastasmātkāyaḥ kalpahīnasvabhāvaḥ // 6 //	BUtam hitvA citta cEtte ca nastas tasmAt kAyaH kalpa hIna svaBAvaH // 6 //
   citta nāsti na dharmāste na kāyo nāpi dhātavaḥ /	citta na asti na DarmAH te na kAyaH na api DAtavaH /
   advayīkaraṇaṃ hīdaṃ tattvaṃ vidvadbhirucyate // 7 //	advayIkaraNam hi idam tattvam vidvadBiH ucyate // 7 //
   anālambamidaṃ sarvamanālambaṃ prabhāṣitam /	anAlambam idam sarvam anAlambam praBAzitam /
   kṛtvā matimanālambāmanālambaṃ samuddhitaṃ // 8 //	kRtvA matim anAlambAm anAlambam samudDitam // 8 //
   dānaśīlakṣamāvīryadhyānādau suniṣevite /	dAna SIla kzamA vIrya DyAna AdO su nizevite /
   acireṇaiva kālena paramāṃ bodhimāpsyati // 9 //	acireNa eva kAlena paramAm boDim Apsyati // 9 //
   upāyaprajñayosthitvā sattvāṃśca karuṇāpayet /	upAya prajQayA usTitvA sattvAn= ca karuNApayet /
   sarvajñānaṃ śighrameva lapsyate nahi saṃśayaḥ // 10 //	sarva jQAnam SiGram eva lapsyate na hi saMSayaH // 10 //
   nāmamātramidaṃ sarvaṃ saṃjñāmātre pratiṣṭhitam /	nAma mAtram idam sarvam saMjQA mAtre pratizWitam /
   nābhidhānātpṛthagbhūtamabhidheyaṃ na vidyate // 11 //	na aBiDAnAt pRTak BUtam aBiDeyam na vidyate // 11 //
   anāmakāḥ sarvadharmā nirātmānaḥ prakīrtitāḥ /	anAmakAH sarva DarmAH nirAtmAnaH prakIrtitAH /
   ime dharmā abhūtāśca kalpanāyāḥ samuddhitāḥ /	ime DarmAH aBUtAH ca kalpanAyAH samudDitAH /
   kalpanā sāpi śūnyeyaṃ yayā śūnyeti kalpitāḥ // 12 //	kalpanA sA api SUnyA iyam yayA SUnya iti kalpitAH // 12 //
   cakṣu paśyati rūpāṇi tattvavaktrā yaducyate /	cakzu paSyati rUpANi tattva vaktrA yat ucyate /
   mithyābhimānalokasya sāṃvṛtaṃ satyamīritam // 13 //	miTyA aBimAna lokasya sAMvRtam satyam Iritam // 13 //
   sāmagryā darśanaṃ yatra prakāśayati nāyakaḥ /	sAmagryAH darSanam yatra prakASayati nAyakaH /
   prāhopacārabhūmiṃ tāṃ paramārthasya buddhimān // 14 //	prAha upacAra BUmim tAm parama arTasya budDimAn // 14 //
   na cakṣuḥ prekṣate rūpaṃ mano dharmānnvetti ca /	na cakzuH prekzate rUpam manaH DarmAn nvetti ca /
   etattu paraṃ satyaṃ lokasya viṣayo na yat // 15 //	etat tu param satyam lokasya vizayaH na yat // 15 //
   cakṣurnāsti na rūpañca dṛśyaṃ nāsti na manaskriyā /	cakzuH na asti na rUpaQca dRSyam na asti na manaskriyA /
   cittaṃ svapnasamaṃ bhrāntiḥ sarvaṃ na sadasanna ca // 16 //	cittam svapna samam BrAntiH sarvam na sat asat na ca // 16 //
   ācārya āryanāgārjunakṛta bhavasaṅkrāntiparikathā sampūrṇā //	AcAryaH Arya nAgArjuna kRta Bava saFkrAnti parikaTA sampUrNAH //
