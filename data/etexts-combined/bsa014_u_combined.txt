   Śikṣāsamuccayakārikā (Śsk)	ŚikzA samuccaya kArikA (Śsk)
   yadā mama pareṣāṃ ca bhayaṃ duḥkhaṃ ca na priyam /	yadA mama parezAm ca Bayam duHKam ca na priyam /
   tadātmanaḥ ko viśeṣo yattaṃ rakṣāmi netaram // Śsk_1 //	tad A AtmanaH kaH viSezaH yattam rakzAmi na itaram // Śsc _1 //
   duḥkhāntaṃ kartukāmena sukhāntaṃ gantumicchatā /	duHKAntam kartu kAmena suKa antam gantum icCatA /
   śraddhāmūlaṃ dṛḍhīkṛtya bodhau kāryā matirdṛḍhā // Śsk_2 //	SradDA mUlam dRXIkRtya boDO kAryA matiH dRXA // Śsc _2 //
   (śikṣādaro) mahāyānādbodhisattvasya saṃvaraḥ /	(SikzA daro) mahAyAnAt boDisattvasya saMvaraH /
   marmasthānānyato vidyādyenānāpattiko bhavet // Śsk_3 //	marma sTAnAni atas vidyA Adi ena anApattikaH Bavet // Śsc _3 //
   ātmabhāvasya bhogānāṃ tryadhvavṛtteḥ śubhasya ca /	Atma BAvasya BogAnAm tri aDva vRtteH SuBasya ca /
   utsargaḥ sarvasattvebhyastadrakṣā śuddhivardhanam // Śsk_4 //	utsargaH sarva sattveByaH tad rakzA SudDi varDanam // Śsc _4 //
   paribhogāya sattvānāmātmabhāvādi dīyate /	pariBogAya sattvAnAm Atma BAva Adi dIyate /
   arakṣite kuto bhogaḥ ki dattaṃ yanna bhujyate // Śsk_5 //	arakzite kutas BogaH ki dattam yat na Bujyate // Śsc _5 //
   tasmātsattvopabhogārthamātmabhāvādi pālayet /	tasmAt sattva upaBoga arTam Atma BAva Adi pAlayet /
   kalyāṇamitrānutsargāt sūtrāṇāṃ ca sadekṣaṇāt // Śsk_6 //	kalyANa mitra anutsargAt sUtrANAm ca sadA IkzaNAt // Śsc _6 //
   tatrātmabhāve kā rakṣā yadanarthavivarjanam /	tatra Atma BAve kA rakzA yat anarTa vivarjanam /
   kenaitallabhyate sarvaṃ niṣphalaspandavarjanāt // Śsk_7 //	kena etat laByate sarvam nizPala spanda varjanAt // Śsc _7 //
   etatsidhyetsadā smṛtyā smṛtistīvrādarādbhavet /	etat siDyet sadA smRtyA smRtiH tIvra AdarAt Bavet /
   ādaraḥ śamamāhātmyaṃ jñātvātāpena jāyate // Śsk_8 //	AdaraH Sama mAhAtmyam jQAtvA tApena jAyate // Śsc _8 //
   samāhito yathābhūtaṃ prajānātītyavadanmuniḥ /	samAhitaH yaTABUtam prajAnAti iti avadat muniH /
   śamācca na caleccitaṃ bāhyaceṣṭā nivartanāt // Śsk_9 //	SamAt ca na calet citam bAhya cezwAH nivartanAt // Śsc _9 //
   sarvatrāpacalo mandamatisnigdhābhibhāṣaṇāt /	sarvatra apacalaH manda mati snigDa aBiBAzaNAt /
   āvavarjayejjanaṃ bhavyamādeyaścāpi jāyate // Śsk_10 //	Avavarjayet janam Bavyam AdeyaH ca api jAyate // Śsc _10 //
   anādeyaṃ tu taṃ lokaḥ paribhūya jināṅkuram /	anAdeyam tu tam lokaH pariBUya jinAFkuram /
   bhasmacchanno yathā vahniḥ pacyeta narakādiṣu // Śsk_11 //	Basma= CannaH yaTA vahniH pacyeta naraka Adizu // Śsc _11 //
   ratnameghe jinenoktastena saṅkṣepasaṃvaraḥ /	ratna meGe jinena uktaH tena saFkzepa saMvaraH /
   yenāprasādaḥ sattvānāṃ tadyatnena vivarjayet // Śsk_12 //	yena aprasAdaH sattvAnAm tat yatnena vivarjayet // Śsc _12 //
   eṣā rakṣātmabhāvasya bhaiṣajyavasānādibhiḥ /	ezA rakzA Atma BAvasya BEzajya vasA nAdiBiH /
   ātmatṛṣṇopabhogāttu kliṣṭāpattiḥ prajāyate // Śsk_13 //	Atma tRzNA upaBogAt tu klizwa ApattiH prajAyate // Śsc _13 //
   sukṛṭārambhiṇā bhāvyaṃ mātrajñena ca sarvataḥ /	su kRwa AramBiNA BAvyam mAtrajQena ca sarvatas /
   iti śikṣāpadādasya bhogarakṣā na duṣkarā // Śsk_14 //	iti SikzA padAt asya Boga rakzA na duzkarAH // Śsc _14 //
   svārthavipāka vaitṛṣṇyācchubhaṃ saṃrakṣitaṃ bhavet /	sva arTa vipAka vEtRzNyAt SuBam saMrakzitam Bavet /
   paścātāpaṃ na kurvīta na ca kṛtvā prakāśayet // Śsk_15 //	paScAtApam na kurvIta na ca kRtvA prakASayet // Śsc _15 //
   lābhasatkārabhītaḥ syādunnatiṃ varjayetsadā /	lABa satkAra BItaH syAt unnatim varjayet sadA /
   bodhisattvaḥ prasannaḥ syāddharme vimatimutsṛjet // Śsk_16 //	boDisattvaH prasannaH syAt Darme vimatim utsRjet // Śsc _16 //
   śodhitasyātmabhāvasyabhogaḥ pathyo bhaviṣyati /	SoDitasya Atma BAvasya BogaH paTyaH Bavizyati /
   samyaksiddhasya bhaktasya niṣkaṇasyeva dehinām // Śsk_17 //	samyak sidDasya Baktasya nizkaNasya iva dehinAm // Śsc _17 //
   tṛṇacchanna yathā śasyaṃ rogaiḥ sīdati naidhate /	tRNa= Canna yaTA Sasyam rogEH sIdati na eDate /
   buddhāṅkurastathā vṛddhiṃ kleśacchanno na gacchati // Śsk_18 //	budDa aFkuraH taTA vRdDim kleSa= CannaH na gacCati // Śsc _18 //
   ātmabhāvasya kā śuddhiḥ pāpakleśaviśodhanam /	Atma BAvasya kA SudDiH pApa kleSa viSoDanam /
   sambuddhottayarthasāreṇa yatnabhāve tvapāyagaḥ // Śsk_19 //	sambudDa uttayarTa sAreNa yatna BAve tu apAya gaH // Śsc _19 //
   kṣameta śrutameṣeta saṃśrayeta vanaṃ tataḥ /	kzameta Srutam ezeta saMSrayeta vanam tatas /
   sāmādhānāya yujyeta bhāvayedaśubhādikam // Śsk_20 //	sAma ADAnAya yujyeta BAvayet aSuBa Adikam // Śsc _20 //
   bhogaśuddhiṃ ca jānīyātsamyagājīvaśodhanāt /	Boga SudDim ca jAnIyAt samyak AjI va SoDanAt /
   śūnyatākaruṇāgarbheceṣṭitātpuṇyaśodhanam // Śsk_21 //	SUnyatA karuNA garBeca izwitAt puNya SoDanam // Śsc _21 //
   grahītāraḥ subahavaḥ svalpaṃ ceda manena kim /	grahItAraH su bahavaH su alpam ca ida manena kim /
   na cātitṛptijanakaṃ vardhanīyamidaṃ tataḥ // Śsk_22 //	na ca ati tRpti janakam varDanIyam idam tatas // Śsc _22 //
   ātmabhāvasya kā vṛddhirbalānālasyavardhanam /	Atma BAvasya kA vRdDiH balAnAlasya varDanam /
   śūnyatākaruṇāgarbhāddānādbhogasya vardhanam // Śsk_23 //	SUnyatA karuNA garBAt dAnAt Bogasya varDanam // Śsc _23 //
   kṛtvādāveva yatnena vyavasāyāśayau dṛḍhau /	kRtvA AdO= eva yatnena vyavasAya ASayO dRXO /
   karuṇāṃ ca puraskṛtya yateta śubhavṛddhaye // Śsk_24 //	karuNAm ca puraskRtya yateta SuBa vRdDaye // Śsc _24 //
   bhadracaryāvidhiḥ kāryo vandanādiḥ sahādarāt /	Badra caryA viDiH kAryaH vandana AdiH saha AdarAt /
   śraddhādīnāṃ sadābhyāso maitrī buddhādyanusmṛtiḥ // Śsk_25 //	SradDA AdInAm sadA aByAsaH mEtrI budDa Adi anusmRtiH // Śsc _25 //
   sarvāvasthāsu sattvārtho dharmadānaṃ nirāmiṣam /	sarva avasTAsu sattva arTaH Darma dAnam nirAmizam /
   bodhicittaṃ ca puṇyasya vṛddhihetuḥ samāsataḥ // Śsk_26 //	boDi cittam ca puNyasya vRdDi hetuH samAsatas // Śsc _26 //
   siddhiḥ samyakprahāṇānāmapramādāviyojanāt /	sidDiH samyak prahANAnAm apramAda aviyojanAt /
   smṛtyātha samprajanyena yoniśaścintanena ca // Śsk_27 //	smRtyA aTa samprajanyena yoniSas cintanena ca // Śsc _27 //
