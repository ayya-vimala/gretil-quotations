   Satyadvayāvatāranāma (Sda)	Satya dvaya avatAra nAmaH (Sda)
   // namo mahākāruṇikāya //	// namaH mahA kAruNikAya //
   dve satye samupāśritya buddhānāṃ dharmadeśanā /	dve satye samupASritya budDAnAm Darma deSanA /
   lokasaṃvṛtisatyaṃ ca satyaṃ ca paramārthataḥ // Sda_1 //	loka saMvRti satyam ca satyam ca paramArTataH // SdaH _1 //
   saṃvṛttirmanyate dvedhā mithyā ca tathatā tathā /	saMvRttiH manyate dveDA miTyA ca taTatA taTA /
   ādyā dvidhodacandraśca kusiddhāntavitarkaṇā // Sda_2 //	AdyAH dviDoda candraH ca kusidDAnta vitarkaNAH // SdaH _2 //
   avicāraikaramyā ca vināśotpādadharmiṇī /	avicAra eka ramyA ca vinASa utpAda DarmiNI /
   arthakriyāsamarthā ca tathatāsaṃvṛtirmatā // Sda_3 //	arTa kriyA samarTA ca taTatA saMvRtiH matA // SdaH _3 //
   eka evaparo hyarthaḥ paraiśca dvividho mataḥ /	ekaH eva paraH hi arTaH parEH ca dviviDaH mataH /
   na kācid dharmatā siddhā kuto dvitryādikaṃ bhavet // Sda_4 //	na kAcid DarmatA sidDA kutas dvi tri Adikam Bavet // SdaH _4 //
   anutpādanirodhādi deśanāvākyalakṣitam /	anutpAda niroDa Adi deSanA vAkya lakzitam /
   paramārthābhinnaśīlatvān naiva dharmā na dharmatā // Sda_5 //	parama arTa aBinna SIla tvAt na eva DarmAH na DarmatA // SdaH _5 //
   śūnyatāyāṃ vibhedastu kiñcinmātraṃ na vidyate /	SUnyatAyAm viBedaH tu kiQcit mAtram na vidyate /
   nirvikalpatayā bodhe śūnyatādṛṣṭirucyate // Sda_6 //	nirvikalpa tayA boDe SUnyatA dRzwiH ucyate // SdaH _6 //
   uktaṃ sūtre sugambhīre taddarśanamadarśanam /	uktam sUtre su gamBIre tad darSanam adarSanam /
   tatra draṣṭā na dṛśyaṃ cāpyanādinidhanaṃ śivam // Sda_7 //	tatra drazwA na dRSyam ca api anAdi niDanam Sivam // SdaH _7 //
   nirvikalpaṃ nirālambaṃ bhāvābhāvavivarjitam /	nirvikalpam nirAlambam BAva aBAva vivarjitam /
   anāśrayāpratiṣṭhānam atulyaṃ nirgatāgatam // Sda_8 //	anASraya apratizWAnam atulyam nirgata Agatam // SdaH _8 //
   anirvācyamanābhāsaṃ nirvikāramasaṃskṛtam /	anirvAcyam anABAsam nirvikAram asaMskRtam /
   yogigamyamidaṃ kleśajñeyāvaraṇavarjitam // Sda_9 //	yogi gamyam idam kleSa jQeya AvaraNa varjitam // SdaH _9 //
   pratyakṣamanumānañca taddvayaṃ bauddhasammatam /	pratyakzam anumAnaQca tad dvayam bOdDa sammatam /
   ubhābhyāṃ śūnyatā gamyetyarvāgdṛṅmohabhāṣitam // Sda_10 //	uBAByAm SUnyatA gamya iti arvAk dRS moha BAzitam // SdaH _10 //
   prasajyeddharmatājñānaṃ tīrthike śrāvake 'pi ca /	prasajyet Darma tA jQAnam tIrTike SrAvake api ca /
   vijñānināñca kiṃ vācyaṃ mādhyamike 'viruddhatā // Sda_11 //	vijQAninAQca kim vAcyam mADyamike avirudDa tA // SdaH _11 //
   tarhi sarve 'pi siddhāntā mānameyatayā samāḥ /	tarhi sarve api sidDAntAH mAnameya tayA samAH /
   sarvatarkaviruddhatvān mānameyāpi dharmatā // Sda_12 //	sarva tarka virudDa tvAt mAnameya api DarmatA // SdaH _12 //
   bāhulyena kathaṃ na syāt pratyakṣaṃ cānumā vṛthā /	bAhulyena kaTam na syAt pratyakzam ca anumA vRTA /
   tīrthyavādanivṛtyarthaṃ vidvadbhiḥ kṛtayaḥ kṛtāḥ // Sda_13 //	tIrTya vAda nivRti arTam vidvadBiH kRtayaH kRtAH // SdaH _13 //
   savikalpāvikalpābhyāṃ jñānābhyāṃ nāvagamyate /	sa vikalpa avikalpAByAm jQAnAByAm na avagamyate /
   āgame 'pi sphuṭaṃ vidvānācāryo bhavya āha ca // Sda_14 //	Agame api sPuwam vidvAn AcAryaH BavyaH Aha ca // SdaH _14 //
   śūnyatāvagatā kena vyākṛtā yā tathāgataiḥ /	SUnyatA avagatA kena vyAkRtA yA taTAgatEH /
   nāgāntevāsicandro hi dharmatāsatyadarśakaḥ // Sda_15 //	nAga antevA asi candraH hi DarmatA satya darSakaH // SdaH _15 //
   tataḥ paramparāmnāyairdharmatāsatyagamyatā /	tatas paramparAmnAyEH DarmatA satya gamya tA /
   dharmaskandhasahasrāṇi catvāryuktānyaśīti ca // Sda_16 //	Darma skanDa sahasrANi catvAri uktAnyaSIti ca // SdaH _16 //
   dharmatāntargataṃ sarvam muktistu śūnyatābodhes	DarmatA antargatam sarvam muktiH tu SUnyatA boDeH 
   tadarthā śeṣabhāvanā tathyasaṃvṛtimādhūya // Sda_17 //	tad arTA Seza BAvanA taTya saMvRti mADUya // SdaH _17 //
   śūnyatābhyasane sati saṃvṛtihetoḥpuṇyādeḥ /	SUnyatA aByasane sati saMvRti hetoH puNya AdeH /
   paralokācca vañcyate viviktārthamajānānaḥ // Sda_18 //	para lokAt ca vaQcyate vivikta arTam ajAnAnaH // SdaH _18 //
   svalpaśrutisamāśritaḥ yo naraḥ puṇyakṛnnāsti /	su alpa Sruti samASritaH yaH naraH puNya kRt na asti /
   naṣṭaḥ kāpuruṣastu saḥ vināśayati durdṛṣṭā // Sda_19 //	nazwaH kApuruzaH tu saH vinASayati durdRzwAH // SdaH _19 //
   śūnyatā mandamedhasam candrācārya uvācaivam /	SUnyatA manda meDasam candra AcAryaH uvAca evam /
   upāyabhūtaṃ vyavahārasatyamupeyabhūtaṃ paramārthasatyam // Sda_20 //	upAya BUtam vyavahAra satyam upeya BUtam parama arTa satyam // SdaH _20 //
   dvayorvibhedaṃ na hi veda yo sa vrajedapāyaṃ viparītabodhāt /	dvayoH viBedam na hi veda yaH sa vrajet apAyam viparIta boDAt /
   vyavahāramanāgamya paramārtho na deśyate // Sda_21 //	vyavahAram anAgamya parama arTaH na deSyate // SdaH _21 //
   tathyasaṃvṛtisopānamantareṇa vipaścitaḥ /	taTya saMvRti sopAnam antareNa vipaScitaH /
   tattvaprāsādaśikharārohaṇaṃ na hi yujyate // Sda_22 //	tattva prAsAda SiKara ArohaNam na hi yujyate // SdaH _22 //
   yathāyaṃ saṃvṛterbhāso yuktyā kiñcinna labhyate /	yaTA ayam saMvRteH BAsaH yuktyA kiQcid na laByate /
   paramārthastvalabdhatvam ādisaṃsthitadharmatā // Sda_23 //	parama arTaH tu alabDa tvam Adi saMsTita DarmatA // SdaH _23 //
   hetupratyayajanyatvāt saṃvṛtirbhāsavanmatā /	hetu pratyaya janya tvAt saMvRtiH BAsa vat matA /
   ayuktaṃ śodhitum cet kairjalacandrādivinirmitam // Sda_24 //	ayuktam SoDitum ced kEH jala candra Adi vinirmitam // SdaH _24 //
   nānāpratyayajanyatvāt siddho bhāso 'khilastataḥ /	nAnA pratyaya janya tvAt sidDaH BAsaH aKilaH tatas /
   pratyayānāṃ tu vicchedātsaṃvṛttyāpi na sambhavaḥ // Sda_25 //	pratyayAnAm tu vicCedAt saMvRttyA api na samBavaH // SdaH _25 //
   evaṃ dṛṣṭerasaṃmohāj jāte caryāviśodhane /	evam dRzweH asaMmohAt jAte caryA viSoDane /
   unmārge 'gamanaṃ kṛtvākaniṣṭhaṃ sthānamāpnuyāt // Sda_26 //	unmArge agamanam kRtvA kanizWam sTAnam ApnuyAt // SdaH _26 //
   āyuṣyamalpaṃ bāhulāśca vidyā āyuḥpramāṇaṃ ca kiyanna vidmaḥ /	Ayuzyam alpam bAhulAH ca vidyA AyuH pramANam ca kiyat na vidmaH /
   svābhīṣṭameva pratilambhanīyaṃ haṃsairyathā kṣīramivāmbumadhyāt // Sda_27	sva aBIzwam eva pratilamBanIyam haMsEH yaTA kzIram iva ambu maDyAt // SdaH _27
   arvāgdṛśā mohavaśena cāpi kṣamo na satyadvayanirṇaye 'pi /	arvAk dRSA moha vaSena ca api kzamaH na satya dvaya nirNaye api /
   uktīrgurūṇāmiha sampratītya nyastaṃ dvayaṃ nāgamataṃ hi satyam // Sda_28	uktIH gurUNAm iha sampratItya nyastam dvayam nAga matam hi satyam // SdaH _28
   rājño 'nurodhena suvarṇadvīpe kṛte 'tra śrāddho yadi vā jano 'dya /	rAjQaH anuroDena suvarNa dvIpe kRte atra SrAdDaH yadi vA janaH adya /
   gṛhṇātu samyak suparīkṣaṇena na śraddhayā naiva ca gauraveṇa // Sda_29 //	gRhNAtu samyak su parIkzaNena na SradDayA na eva ca gOraveNa // SdaH _29 //
   sauvarṇarājena guroḥphalena saṃpreṣito devamatirhi bhikṣuḥ /	sOvarNa rAjena guroH Palena saMprezitaH devamatiH hi BikzuH /
   tasyāgrahāt satyadvayāvatāro yuktyā sudhībhistviha vīkṣaṇīyaḥ // Sda_30 //	tasya agrahAt satya dvaya avatAraH yuktyA suDIBiH tu iha vIkzaNIyaH // SdaH _30 //
   ācāryadīpaṅkaraśrījñānaviracitaḥ satyadvayāvatāraḥ samāptaḥ /	AcArya dIpaFkara SrI jQAna viracitaH satya dvaya avatAraH samAptaH /
   tenaiva paṇḍitena anuvādakena vīryasiṃhena cānūdya saṃśodhya ca nirṇītaḥ /	tena eva paNxitena anuvAdakena vIrya siMhena ca AnUdya saMSoDya ca nirNItaH /
