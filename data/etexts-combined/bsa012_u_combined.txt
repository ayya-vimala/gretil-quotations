   Pratītyasamutpādahṛdayakārikā (Pshk)	PratIti asamutpAda hRdaya kArikA (Pshk)
   nāgārjunakṛtā	nAgArjuna kRtA
   dvādaśa ye 'ṅgaviśeṣā muninoddiṣṭāḥ pratītyasambhūtāḥ /	dvAdaSa ye aFga viSezAH muninA uddizwAH pratItya samBUtAH /
   te kleśakarmaduḥkheṣu saṅgṛhītāstriṣu yathāvat // Pshk_1 //	te kleSa karma duHKezu saFgRhItAH trizu yaTAvat // Pshc _1 //
   ādyāṣṭamanavamāḥ syuḥ kleśāḥ karma dvitīyadaśamau ca /	Adya azwamana vamAH syuH kleSAH karma dvitIya daSamO ca /
   śeṣāḥ sapta ca duḥkhaṃ trisaṅgrahā dvādaśa tu dharmāḥ // Pshk_2 //	SezAH sapta ca duHKam tri saFgrahAH dvAdaSa tu DarmAH // Pshc _2 //
   tribhyo bhavati dvandvaṃ dvandvātprabhavanti sapta saptabhyaḥ /	triByaH Bavati dvandvam dvandvAt praBavanti sapta saptaByaH /
   traya udbhavanti bhūyastadeva[tu] bhramati bhavacakram // Pshk_3 //	trayaH udBavanti BUyas ta deva[tu] Bramati Bava cakram // Pshc _3 //
   hetuphalañca[hi]sarvaṃ jagadanyo nāsti kaścidiha sattvaḥ /	hetu PalaQca[hi] sarvam jagat anyaH na asti kaScid iha sattvaH /
   śūnyebhya eva śūnyā dharmāḥ prabhavanti dharmebhyaḥ // Pshk_4 //	SUnyeByaH eva SUnyAH DarmAH praBavanti DarmeByaH // Pshc _4 //
   svādhyāyadīpamudrādarpaṇaghoṣa 'rkakāntabījāmlaiḥ /	svADyAya dIpa mudrA darpaNa Goza arka kAnta bIja amlEH /
   skandhapratisandhirasaṅkramaśca vidvadbhiravadhāryau // Pshk_5 //	skanDa pratisanDiH asaFkramaH ca vidvadBiH avaDAryO // Pshc _5 //
   ya ucchedaṃ prakalpayatyatisūkṣme 'pi vastuni /	yaH ucCedam prakalpayati ati sUkzme api vastuni /
   pratītyasambhavasyārthamavijñaḥ sa na paśyati // Pshk_6 //	pratIti asamBavasya arTam avijQaH sa na paSyati // Pshc _6 //
   nāpaneyamataḥ kiñcit prakṣepyaṃ nāpi kiñcana /	na apaneyam atas kiQcid prakzepyam na api kiQcana /
   draṣṭavyaṃ bhūtato bhūtaṃ bhūtadarśī vimucyate // Pshk_7 //	drazwavyam BUtataH BUtam BUta darSI vimucyate // Pshc _7 //
   pratītyasamutpādahṛdayakārikā ācārya nāgārjunakṛtā samāptā /	pratItyasamutpAda hRdaya kArikA AcArya nAgArjuna kRtA samAptAH /
